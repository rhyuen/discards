  • A Chalk Outline

    Attempt two at using background-blend-mode in CSS.

    I had to use different elements for the image transforms and the text.

    If I used the text and the image transform in the same element, then the transforms applied to the text as well.

    So, the parent element to both the image and the text is 'position: relative' and this text-container is 'position: absolute' and centered using 'top: 50%', 'left: -50%' and 'transform: translate(-50%, -50%)'.
