Some notes

aria-label = 'for adding a description to a tag'

You can add a title="text that can be seen on a button on hover".

Hide SVGs with aria-hidden="true".

Use aria-label="to describe what the button does".

You can use the :active{} pseudo class on a button and have it transform: translateY(2%) to 'simulate' a depressing effect for buttons.

Issues

The buttons on the bottom left were oblong shaped until I set an explicit width and height.

The SVG Line element was uncoloured until I put and explicit = stroke="white" attribute on the line elements for the plus sign.
