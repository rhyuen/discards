Accessibility

  1. Labels for Inputs by using 'for=idforinput' on 'label' and 'id=idforinput' on 'input'
  2. You can bind inputs to validation rules by using 'aria-describedby'. In the 'input', you use 'aria-describedby="validation-id-name"' and in the validation element, like 'span', you use 'id="validation-id-name"'.
  3. In 'input', use 'aria-live="assertive"' to have a screen reader announce the error when the application state arrives there.

Other stuff

  1. In 'input', you can use 'required' by itself as an attribute to make the value not be empty.
  2. 'minlength="10"' to set the minimum number of characters to 10 of whatever you set the value to be.
