Avoiding Layout Shift

Notes and References

Made on Feb.14, 2024

Why? Layout shift was happening and I didn't know how to address it in a previous discard.

I read most of this Smashing Magazine Article on Layout shift.

What I took away from it was the following code:

img{ inline-size: 100%; block-size: auto; aspect-ratio: 16/9; }

I got the link for the Smashing Magazine article from here (Kent C Dodds).

I also looked around this web.dev article. However, there were a lot of words so I only read some of it, not all of it.

img { width: 100%; height: auto; } img[Attributes Style] { aspect-ratio: auto 640 / 360; }

The above snippets of code were the important bits for me. However, I don't know if you can do the [Attributes Style] in the second one as it looks odd and I've never tried it.
