An attempt at trying nested divs with rounded borders.

Following the example as demonstrated here.

The key is making the child border-radius equal to the parent's border-radius but removing the padding of the parent from the parent's border-radius.

For some reason, there's a thicker padding on the bottom of the nested items with an image in them than the ones without an image in them.

The answer to the above problem is here.

One must set the 'vertical-align' for that child element to 'middle'.

Alternatively, one can also set the 'display' value to 'block'.
