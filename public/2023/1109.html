Pulled gradient from this CodePen.

I didn't know how to do gradient text or, if I did, I forgot. I lifted the code from here (GeeksforGeeks). However, the original impetus for this was wanting text with an image embedded in the text.

                    p{
                        background: var(--GRADIENT);
                        -webkit-text-fill-color: transparent;
                        -webkit-background-clip: text;
                    }

The above lines are the points of interest for making gradient text.

I also had to look up gradient drop shadows for the back button at the top. I used (GeeksforGeeks) as reference again.
