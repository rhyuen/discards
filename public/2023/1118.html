Transparent text with only the text outline from StackOverflow.

I don't always set a background-color for images, but this time I did and noticed that, once again, images did not fill their container 100% again. I recalled solving this issue previously in another Discard. It was resolved with using display:block on the img tag css selector. The reason, if I recall correctly is that text usually has space on the bottom for characters that extend below the line like the character 'y'. This had been extended to images and I guess display: block resolves the issue in some way.
