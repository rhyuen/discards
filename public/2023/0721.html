A Brief Description

Uses keyframes and boxes to create a loading indicator.

Each of the boxes has a slightly different animation duration and transform distance to create the staggered effect.

            animation: bounce 
                var(--animation-duration) 
                infinite 
                alternate 
                cubic-bezier(.2, .65, .6, 1);

If you take out the alternate keyword the animation becomes all stuttery.
