Page Bumpers with Pseudo Elements

An attempt at bumpers with fewer elements.
If you scroll down and go over the limit of the page then you can see a bumper or sorts and the screen will snap back to the edge of the content.
This is the second of two attempts.
This one uses ::before and ::after elements to create the gutters/bumpers that scroll back to the content.
The other uses explicit elements. Both pseudo-elements and explicit elements have no scroll-snap-align values unlike the sibing for the explicit elements and the parent for the pseudo elements.
The scroll-snap-type of "center" is what is responsible for the snapping action.
